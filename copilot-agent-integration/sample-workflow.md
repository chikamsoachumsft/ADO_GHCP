## The Story: US-301 — Transfer between own accounts

### Starting Point (ADO Work Item)

```
Title:    Transfer between own accounts
Type:     User Story
Parent:   FEAT-06 (Internal Transfers)
Points:   8
Priority: 1
State:    Ready for Development
Tags:     transfers; internal
```

**Description:**
> As a customer, I want to transfer money between my own accounts so that I can manage my funds across accounts.

**Acceptance Criteria:**
> Given a customer with 2+ accounts  
> When they select source and destination accounts and enter an amount  
> Then the transfer processes in real-time  
> And both account balances update immediately  
>  
> Given a transfer amount exceeding available balance  
> When submitted  
> Then an error states 'Insufficient funds' and the transfer is blocked  
>  
> Given a successful transfer  
> Then a confirmation shows: amount, from account, to account, timestamp, confirmation number

## Step 2: Copilot Coding Agent Processes the Issue

Copilot reads the issue and the `.github/copilot-instructions.md`, then:

1. **Analyzes the codebase** — finds existing patterns, similar endpoints
2. **Plans the implementation** — identifies files to create/modify
3. **Generates code** — backend endpoint, service logic, DTO, tests
4. **Creates a branch** — `copilot/issue-42-internal-transfer`
5. **Opens a PR** with the implementation

### Sample Generated Code:

**TransfersController.cs** (generated by Copilot)
```csharp
[ApiController]
[Route("api/v1/transfers")]
[Authorize]
public class TransfersController : ControllerBase
{
    private readonly ITransferService _transferService;
    private readonly IAuditLogger _auditLogger;

    public TransfersController(ITransferService transferService, IAuditLogger auditLogger)
    {
        _transferService = transferService;
        _auditLogger = auditLogger;
    }

    /// <summary>
    /// Transfer funds between customer's own accounts.
    /// AC-1: Processes real-time, balances update immediately.
    /// AC-2: Returns 400 if insufficient funds.
    /// AC-3: Returns confirmation with amount, accounts, timestamp, confirmation number.
    /// </summary>
    [HttpPost("internal")]
    public async Task<ActionResult<TransferConfirmation>> CreateInternalTransfer(
        [FromBody] InternalTransferRequest request)
    {
        var customerId = User.GetCustomerId();

        // AC-2: Validate sufficient funds
        var result = await _transferService.CreateInternalTransferAsync(customerId, request);

        if (!result.IsSuccess)
        {
            return BadRequest(new { error = result.ErrorMessage }); // "Insufficient funds"
        }

        // Audit log
        await _auditLogger.LogAsync("InternalTransfer", customerId, new
        {
            request.FromAccountId,
            request.ToAccountId,
            request.Amount,
            result.Value.ConfirmationNumber
        });

        // AC-3: Return confirmation
        return Ok(result.Value);
    }
}
```
